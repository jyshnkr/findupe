# examples/ — demo dataset provenance

Every image and document under `examples/` is synthetically generated for this
repository — procedurally-drawn gradient landscapes (sky, water, dune, and
mountain scenes), not photographs. There is no third-party or stock-photo
content here, so there's nothing to attribute: all files are original work,
dedicated to the public domain under **CC0 1.0**
(https://creativecommons.org/publicdomain/zero/1.0/). Use them for anything,
including your own `findupe` experiments.

## What each file demonstrates

| Files | Tier exercised |
|---|---|
| `inbox/sunset.jpg`, `inbox/sunset copy.jpg` | **Exact** — byte-identical copy |
| `inbox/harbor.jpg`, `inbox/harbor-edit.jpg` | **Strong visual** — resized + re-compressed re-export of the same image (pHash/dHash ≈ 0) |
| `inbox/dunes.jpg`, `inbox/dunes.heic` | **Cross-format sibling** — same photo, JPEG + HEIC; informational only, never a deletion candidate |
| `backup/mountain.jpg` | Control — a genuinely different photo; proves the tool doesn't flag everything |
| `inbox/notes.txt`, `inbox/notes copy.txt` | Non-image exact duplicate — exercises `report-other.html` |

Generated by a one-off script (not committed — plain Pillow drawing calls
plus `pillow-heif` for the HEIC export). Regenerating them exactly isn't
required; treat the committed files as the source of truth.
